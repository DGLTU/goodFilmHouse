## Installing

    npm install
    npm run lint
    npm run dev
    
## 主要功能

#### Client Side
##### 1.瀏覽電影介紹
##### 2.瀏覽今日電影時刻表
##### 3.瀏覽最新消息、FAQ、關於我們、票價資訊
##### 4.依照日期訂購電影票
##### 5.訂購頁面
 + 選擇票種
 + 選擇座位
 + 填寫訂購人資訊、確認訂單資訊
 + 完成訂單、取得訂單序號
##### 6.註冊、登入會員
##### 7.登入會員平台、查看訂票紀錄、會員資料
